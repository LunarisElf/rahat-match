ตั้งค่า Firebase

ทำครั้งเดียวก่อน deploy

⚠ ยังไม่ได้ตั้งค่า Firebase

เปิดไฟล์ index.html ในเครื่อง แล้วหา section FIREBASE CONFIG ในส่วน <script> ด้านล่าง paste ค่าจาก Firebase Console

วิธีรับค่า Firebase config:

  1. ไปที่ firebase.google.com → Get Started → สมัครฟรี (Google)
  2. Add Project → ตั้งชื่อ → Continue → Continue → Create project
  3. Build → Firestore Database → Create database → "Start in test mode" → Next → เลือก location (เช่น asia-southeast1) → Enable
  4. กลับมาหน้า Project Overview → คลิกไอคอน </> Web → ลงทะเบียน app → Copy firebaseConfig object
  5. Paste ลงในไฟล์ index.html ตรง section FIREBASE CONFIG → Save → upload ใหม่

ตำแหน่งใน source: หาคำว่า FIREBASE_CONFIG ในไฟล์ index.html (ประมาณบรรทัด 270)
